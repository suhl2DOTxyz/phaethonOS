FREQUENTLY ASKED QUESTIONS

HOW DO I UPDATE THE SYSTEM?

Since we are a rolling release distribution, simply open Konsole and run the standard package updates using Pacman or yay.

CAN I USE PROPRIETARY NVIDIA DRIVERS?

Yes. Custom NVIDIA packages are maintained within our repositories. Open system settings or use terminal utilities to set up proprietary modules.

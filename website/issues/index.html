[OPEN] ISSUE #081 // LAST UPDATE: 2026-05-24

NVIDIA WAYLAND TEARING UNDER INTEL INTEGRATED FALLBACK

Some hybrid laptops utilizing Intel iGPU + Nvidia dGPU experience display tearing under specific Wayland sessions. Mitigation: Set kernel parameter nvidia-drm.modeset=1 in boot config.

[RESOLVED] ISSUE #079 // LAST UPDATE: 2026-05-24

CALAMARES PRIVILEGED RUNTIME ON WAYLAND LIVE BOOT

Wayland environment variables stripped by standard Polkit escalation blocked Calamares graphical screens. Fixed: Deployed the phaethon-calamares Wayland runner wrapper inside live autostarts.

PRIVACY POLICY

1. ZERO TELEMETRY COMPLIANCE //

PhaethonOS operates under a strict Zero-Telemetry policy. We do not integrate data collection hooks, user behavior monitoring trackers, or remote telemetry scripts inside the live ISO desktop environment or inside the compiled installations. Your operating system, execution logs, and application usages remain strictly local to your computing devices.

2. PACKAGE SYNCHRONIZATION //

All package operations, updates, and kernel compiles leverage direct secure connections to standard official Arch Linux mirrors, CachyOS repository mirrors, or our custom distribution repository servers. These processes log standard network interactions required for updating file signatures and database checksums, and do not track individual users.

3. ENCRYPTION PROTOCOLS //

The Calamares installation script supports advanced full-disk LUKS partitions. We strongly advise developers, system administrators, and privacy advocates to enable LUKS system volumes during installer sequence steps to ensure files and credentials remain secure when system volumes are unmounted.

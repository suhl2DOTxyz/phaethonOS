RELEASE NOTES: BELLE

PHAETHONOS V1.0.0 'BELLE' //

This is the initial stable release of PhaethonOS Belle, custom-tailored with the high-performance BORE kernel scheduler and styled entirely in the Zenless Zone Zero (ZZZ) visual theme.

KEY HIGHLIGHTS

  • Direct Installer Autostart: Boots directly into a Wayland session and launches a graphical, ZZZ-themed Calamares installer sequence.
  • Performance Micro-Schedulers: Injects ananicy-cpp rules and gaming parameters to maximize frame rates and minimize desktop latency.
  • Themed GRUB Bootloader: Complete custom GRUB boot theme centering the distro's gold star logo over a flat black background.
  • Fish & Fastfetch: Interactive shells are pre-configured to output Distro info with a gold ASCII star emblem.
